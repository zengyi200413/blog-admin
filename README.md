# Inkstone 博客后台管理系统

这是一个无需安装依赖、打开 `index.html` 即可运行的博客后台管理系统原型，适合快速演示后台界面、管理流程和前端交互。

## 功能

- 管理员登录
- 仪表盘统计
- 文章新增、编辑、删除、搜索
- 分类与标签管理
- 评论审核
- 站点设置
- 使用 `localStorage` 持久化数据

## 默认账号

- 用户名：`admin`
- 密码：`123456`

## 使用方式

1. 进入项目目录：`E:\work\blog-admin`
2. 直接双击 `index.html`，或用任意静态服务器打开

## 文件结构

- `index.html`：页面结构
- `styles.css`：界面样式
- `app.js`：业务逻辑和本地数据管理
